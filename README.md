A lightweight plugin for Minecraft for collectibles!

## Features
- 🔥 **Lightweight & Fast**
- ✏️ **Full** Customizability!
- ☀️ Supports [**MiniMessage Formatting**](https://webui.advntr.dev/)

The plugin adds tools for making finds.
Here you will find everything you need to create a working find system.
The plugin is fully configurable.
The config has comments that tell you what it is for.

Plugin works on 1.21.x, soon will be updated to 26.1.
